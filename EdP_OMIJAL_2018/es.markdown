# Descripción

Tu tarea es fácil. Créeme. Podría ser MUCHO más complicada.
Dado un entero $ n $, deberás encontrar el número de divisores de $ n! \mod 987654321 $

# Entrada

Un entero $ n $

# Salida

El número de divisores de $ n! \mod 987654321 $
# Ejemplo

||input
10
||output
270
||end

# Límites

$ 1 \leq n \leq 1000 $
